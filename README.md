# gas-leaking-project-using-Arduino-uno-
An Arduino  using gas leakage project without GSM &amp; LCD display gas sensor to detect leaks and buzzer alarm when gas connection exceeds a safe limit. This simple setup provides immediate audible warning  for gas leaks using only the Arduino, sensor, and buzzer
